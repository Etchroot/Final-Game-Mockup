# 마녀의 향수 공방 (Witch Perfume Lab)

## 1. 프로젝트 개요

마녀의 향수 공방은 사용자가 제시된 감정이나 분위기 문장에 어울리는 향을 선택하여 향수를 제작하는 웹 기반 미니게임이다.

플레이어는 10가지 기본 향 중에서 총 10번의 선택을 통해 향을 조합하며, 완성된 향수의 레시피와 분위기를 확인할 수 있다.  
결과는 OpenAI API를 통해 생성된 감성적인 향 설명과 함께 제공된다.

이 프로젝트는 **설치 없이 웹 브라우저에서 바로 플레이할 수 있는 캐주얼 미니게임**을 목표로 한다.

---

# 2. 프로젝트 목표

- 웹 브라우저에서 즉시 플레이 가능한 미니게임
- 1회 플레이 시간 1~3분
- 감성적인 결과 생성
- SNS 공유 가능한 결과 카드
- 향수 구매 페이지로 연결 가능한 구조

---

# 3. 핵심 게임 컨셉

플레이어는 **마녀의 공방에서 향수를 만드는 마법사 역할**을 한다.

제시된 문장에 어울리는 향을 선택해 항아리에 넣으면  
마녀가 그 향을 조합해 하나의 향수로 완성한다.

이 게임은 정답을 맞히는 게임이 아니라  
**감정을 해석하는 창작형 게임**이다.

---

# 4. 핵심 플레이 루프

1. 게임 시작
2. 랜덤 문장 제시
3. 플레이어가 향 선택
4. 항아리에 향이 추가됨
5. 총 10번 선택
6. 향수 완성
7. 결과 화면 표시

결과 화면에는 다음 정보가 포함된다.

- 향수 이름
- 향 레시피 비율
- 분위기 키워드
- 향 설명
- 공유 버튼
- 구매 버튼

---

# 5. 문장 데이터

목업에서는 **10개의 문장을 사용한다**.

게임 시작 시 랜덤으로 하나 선택된다.

### 문장 리스트

1. 잔잔한 바람이 불어오는 밤바다의 향  
2. 햇살이 스며드는 이른 아침의 창가  
3. 비가 막 그친 숲길을 걷는 기분  
4. 오래된 편지를 다시 펼쳐보는 순간  
5. 겨울밤 담요 속에서 마시는 따뜻한 우유  
6. 아무도 없는 새벽 골목의 공기  
7. 첫사랑을 떠올리게 하는 봄날의 향기  
8. 조용한 도서관 구석에 앉아 있는 오후  
9. 마법이 시작될 것 같은 보랏빛 황혼  
10. 막 세탁한 흰 셔츠를 입은 날의 느낌  

또한 **문장 변경 토글 UI**를 제공하여 다른 문장으로 바꿀 수 있다.

---

# 6. 향 데이터

게임에는 10가지 기본 향이 존재한다.

| 향 | 감정 | 색상 |
|---|---|---|
| 라벤더 | 평온 | 연보라 |
| 레몬 | 상쾌함 | 노랑 |
| 바닐라 | 따뜻함 | 크림 |
| 소나무 | 자연 | 초록 |
| 장미 | 낭만 | 핑크 |
| 오렌지 | 활력 | 주황 |
| 샌달우드 | 고요 | 베이지 |
| 머스크 | 신비 | 보라 |
| 코튼 | 순수 | 하늘색 |
| 페트리코르 | 그리움 | 회색 |

각 향 버튼에는 다음 요소가 포함된다.

- 향 아이콘
- 향 이름
- 색상

---

# 7. 게임 규칙

- 플레이어는 총 **10번 향을 선택할 수 있다**
- 클릭 1회 = 향수 10%
- 같은 향 여러 번 선택 가능
- 10회 선택 시 자동 완성

### 레시피 계산 예시

선택 결과:

라벤더 3  
바닐라 2  
코튼 5  

결과 레시피

라벤더 30%  
바닐라 20%  
코튼 50%
