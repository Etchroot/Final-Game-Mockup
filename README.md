# Final-Game-Mockup
MS AI 파이널 프로젝트 게임목업

# 마녀의 향수 공방

제시된 문장에 어울리는 향을 조합해 나만의 향수를 만드는 웹 기반 미니게임입니다.

사용자는 10가지 기본 향 중 원하는 향을 총 10번 선택해 향수를 완성할 수 있으며,  
완성된 향수의 레시피와 분위기 설명을 결과 화면에서 확인할 수 있습니다.

## 주요 기능
- 랜덤 문장 제시
- 10가지 향 선택
- 10회 클릭 기반 향수 제작
- 향 조합 비율 계산
- OpenAI 기반 결과 설명 생성
- 결과 공유 기능
- 구매 페이지 연결 버튼

## 목표
- 설치 없이 웹에서 바로 실행
- 짧고 가볍게 즐길 수 있는 감성형 미니게임
- 하루 안에 구현 가능한 MVP 프로토타입 제작

## 기술 방향
- HTML5 기반 웹 게임 스타일
- 브라우저에서 바로 실행 가능한 구조
- MVP 중심의 단순하고 직관적인 UI

## 로컬 실행
1. `npm install`로 의존성 설치
2. `npm run dev`로 개발 서버 실행
3. 브라우저에서 `http://localhost:3000` 접속

상세 내용과 현재 구현 범위는 `docs/overview.md`를 확인하세요.
